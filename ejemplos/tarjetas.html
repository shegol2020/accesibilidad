Comentarios Técnicos:

  • Problema del primer ejemplo: El lector de pantalla dirá: "Enlace, Crisis en el Ártico El deshielo alcanza niveles récord este verano, afectando a la fauna local y al clima global. Leer más". Es demasiada información para un solo enlace. Además la estructura es inválida, ya que no se debe anidar un <button> dentro de un enlace <a>.
  • Ventaja del segundo ejemplo: Se utiliza un área de clic extendida mediante CSS para facilitar la interacción. Además, el uso de la clase .sr-only proporciona un "nombre accesible" completo para los lectores de pantalla ("Leer más sobre las nuevas tecnologías médicas").
